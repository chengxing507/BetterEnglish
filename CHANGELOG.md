# Changelog

## v1.5.0 (2026-07-05)

### ✨ 新功能
- **测验模式** - 导入页新增独立开关，仅 1 次答题机会，不展示正误反馈
- **标记上题** - 练习中可返回标记上一题，标记计数显示在顶部，点击可查看/修改标记列表
- **局域网监看** - 在 App 首页启动 HTTP 服务器，同级局域网内另一设备浏览器打开监看地址，实时查看练习进度
- **管理面板展示正误情况** - 管理面板新增复选框，可控制答题时是否展示正误提示

### 🐛 Bug 修复
- **倒计时在多次机会时消失** - 首次答错后计时器被隐藏的问题，已修正为仅在我答完或超时时停止计时
- **页面卡死无法点击** - updateModeHint 函数缺少闭合大括号导致 JS 解析失败，所有事件失效
- **导出功能不可用** - 历史记录导出改用 Blob + URL.createObjectURL 方式，兼容 Android WebView
- **AI 复核格式异常误判** - 新增 parseAIJson() 多层降级解析，兼容 AI 各种输出格式
- **页面冻结导致导入正常但导出无效** - 回退至稳定版本后重新正确实现改动

### 🔧 其他
- 版本号更新至 v1.5.0
- README 文档更新，新增局域网监看使用说明
- 新增 CHANGELOG.md 更新日志
